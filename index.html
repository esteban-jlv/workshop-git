Control de Versiones y GIT

¿Qué es un control de versiones?

Un control de versiones es un sistema que registra los cambios realizados en un conjunto de archivos a lo largo del tiempo, de modo que puedas recuperar versiones específicas más adelante.

¿Qué es GIT?

GIT es un sistema de control de versiones distribuido, diseñado para manejar desde proyectos pequeños hasta muy grandes con velocidad y eficiencia.

¿Qué es un repositorio?

Un repositorio es un espacio donde se almacenan los archivos y las versiones de un proyecto. Puede ser local (en tu computadora) o remoto (en un servidor).

Repositorio local: Es la copia del repositorio que resides en tu máquina.

Repositorio remoto: Es una copia del repositorio que reside en un servidor externo, como GitHub.

¿Qué es GITHUB?

GitHub es una plataforma en línea para alojar proyectos que utilizan GIT como sistema de control de versiones. Permite colaborar con otros desarrolladores y facilita la gestión de repositorios.

¿Qué es una rama en GIT y para qué sirve?

Una rama en GIT es una línea de desarrollo independiente creada a partir de la rama principal (normalmente llamada master). Sirve para trabajar en nuevas funcionalidades o solucionar problemas sin interferir con el trabajo en la rama principal.

¿Cuáles son los estados en GIT?

Los estados en GIT son:

  • Modificado: El archivo ha sido modificado pero aún no se ha confirmado a la base de datos.
  • Preparado: Los cambios han sido marcados para ser incluidos en la próxima confirmación.
  • Confirmado: Los cambios han sido guardados de manera permanente en la base de datos.

¿Qué es la terminal, consola o línea de comandos y para qué sirve?

La terminal (o consola o línea de comandos) es una interfaz de texto que permite interactuar con el sistema operativo mediante comandos. Sirve para ejecutar tareas como la gestión de archivos, instalación de software, y ejecución de programas, entre otros.

Escribir el funcionamiento de cada uno de los siguientes comandos:

  • ls
    • Función: Lista los archivos y directorios en el directorio actual.
    Uso: Simplemente escribe ls y presiona Enter en la terminal.
    Ejemplo: Muestra una lista de archivos y subdirectorios en el directorio actual.
  • cd "directorio"
    • Función: Cambia el directorio de trabajo actual a uno especificado.
    Uso: Escribe cd seguido del nombre del directorio al que quieres moverte.
    Ejemplo: cd documentos te llevará al directorio llamado "documentos" dentro del directorio actual.
  • cd ..
    • Función: Cambia al directorio padre (directorios superiores) del directorio actual.
    Uso: Simplemente escribe cd .. y presiona Enter.
    Ejemplo:Si estás en /home/usuario/documentos, cd .. te llevará a /home/usuario.
  • pwd
    • Función: Muestra la ruta completa del directorio actual (Print Working Directory).
    Uso: Simplemente escribe pwd y presiona Enter.
    Ejemplo: Muestra la ruta completa del directorio actual, por ejemplo /home/usuario/documentos.
  • mkdir "nombre"
    • Función: Crea un nuevo directorio con el nombre especificado.
    Uso: Escribe mkdir seguido del nombre que deseas darle al nuevo directorio.
    Ejemplo: mkdir proyecto1 creará un nuevo directorio llamado "proyecto1" en el directorio actual.
  • touch "nombre"
    • Función: Crea un nuevo archivo vacío con el nombre especificado.
    Uso: Escribe touch seguido del nombre que deseas darle al nuevo archivo.
    Ejemplo: touch archivo.txt creará un archivo vacío llamado "archivo.txt" en el directorio actual.
  • rm "nombre"
    • Función: Elimina (borra) un archivo.
    Uso: Escribe rm seguido del nombre del archivo que deseas eliminar.
    Ejemplo: rm archivo.txt eliminará el archivo llamado "archivo.txt" en el directorio actual.
  • cp "nombre" "directorio"
    • Función: Copia un archivo en otro directorio.
    Uso: Escribe cp seguido del nombre del archivo que deseas copiar y luego la ruta del directorio de destino.
    Ejemplo: cp archivo.txt /home/usuario/backups copiará el archivo "archivo.txt" al directorio "/home/usuario/backups".
  • mv "nombre" "directorio"
    • Función: Mueve un archivo a otro directorio o le cambia el nombre.
    Uso: Escribe mv seguido del nombre del archivo que deseas mover y luego la ruta del directorio de destino o el nuevo nombre.
    Ejemplo: mv archivo.xt /home/usuario/documentos moverá el archivo "archivo.txt" al directorio "/home/usuario/documentos". mv archivo.txt nuevo_nombre.txt renombrará el archivo "archivo.txt" a "nuevo_nombre.txt" en el mismo directorio.
